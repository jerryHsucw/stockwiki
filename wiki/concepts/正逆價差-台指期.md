# 正逆價差（台指期）

**最後更新：** 2026-06-08
**相關集數：** 2026060801（下集）、多次提及

## 概述

台指期（台灣加權股價指數期貨）的「正逆價差」是觀察市場多空力道的重要指標，節目常以此判斷短期大盤方向。

## 定義

- **正價差（Contango）：** 期貨價格 > 現貨指數 → 市場偏多，預期指數上漲
- **逆價差（Backwardation）：** 期貨價格 < 現貨指數 → 市場偏空，預期指數下跌或空方壓制

## 計算方式

```
價差 = 台指期近月結算價 − 加權指數現貨點數
正值 = 正價差；負值 = 逆價差
```

## 操作意義

- **正價差大** → 多頭信心強，伺機作多
- **逆價差深** → 空方主導，避免追高；6/17 結算前尤其注意壓低結算風險
- **由正轉負** → 盤中或盤後出現轉折，需隔日密切觀察
