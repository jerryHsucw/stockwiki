# 護國神電——AI 電力基礎建設

**最後更新：** 2026-05-27
**相關集數：** 2026052703（上集）

---

## 概述

「護國神電」是節目（錢線百分百）命名的台股主題概念，指台灣廠商在 AI 資料中心電力基礎設施（Power Infrastructure）的核心地位。2026-05-27 起，此概念從原先的 BBU 備援電力擴展為四大方向，涵蓋從發電端到配電端的完整電力鏈。

黃仁勳於 2026-05 COMPUTEX 台灣行程中明確指出：**「缺電是 AI 發展最大瓶頸」**，再次確認電力基建的戰略重要性。

---

## 四大方向

### 1. GaN 氮化鎵功率元件（第三代半導體）

| 項目 | 內容 |
|------|------|
| 技術定位 | 替代矽基功率元件，開關頻率更高、損耗更低、體積更小 |
| 應用場景 | AI 伺服器電源供應器（PSU）、資料中心電力轉換 |
| 台灣代表股 | **嘉晶（3016）**（磊晶片，漢民集團）、**漢磊（3707）**（6吋晶圓代工，漢民集團） |
| 現況 | 外資 2026 年才開始認識；底部低位階；轉虧為盈中 |

### 2. HVDC 高壓直流匯流排

| 項目 | 內容 |
|------|------|
| 技術定位 | 資料中心內部 1500V 高壓直流電力傳輸系統，減少 AC→DC 轉換損耗 |
| 應用場景 | AI 伺服器機架供電、UPS/BBU 整合 |
| 台灣代表股 | **台達電（2308）**（龍頭）、**光寶科（2301）**（高壓直流兇悍）、**貿聯-KY（3665）**（HVDC線束一站式） |
| 現況 | 台達電/光寶科已創歷史新高；貿聯-KY 套牢區等解套 |

### 3. 智慧配電 PDU（Power Distribution Unit）

| 項目 | 內容 |
|------|------|
| 技術定位 | 從傳統「插線板」升級為聯網+遠端控制+即時監控每張 GPU 功耗的智慧裝置 |
| 應用場景 | AI 工廠（GPU 叢集）每台 GPU 耗電不同需智慧分配；輝達 NVLink Fusion AI 工廠標配 |
| 台灣代表股 | **研華（2395）**（工業電腦龍頭，黃仁勳兆元宴座上賓）、**健和興（3017）**（高壓精密連接器） |
| 現況 | 研華確認為黃仁勳兆元宴合作廠商；智慧 PDU 為 AI 工廠「智慧」必要條件 |

### 4. SOFC 固態氧化物燃料電池

| 項目 | 內容 |
|------|------|
| 技術定位 | 固體氧化物燃料電池（Solid Oxide Fuel Cell）：吃氫/天然氣/沼氣，發電效率 55-60%（遠優於傳統30%） |
| 應用場景 | 資料中心現場（On-Site）發電，不需改建整個電網；搭配 BBU 做持續供電解決方案 |
| 台灣代表股 | **中興電（1513）**（美國 Bloom Energy 台灣合作廠；投信 150-155 元系統性買超）、**康舒（6282）**（Bloom Energy SOFC 合作） |
| 現況 | SOFC 全球市場年複合成長 24.5%（至 2035）；歐美 2020/2021 起制定規格標準 |

## 節目觀點：電力鏈整體邏輯

```
[發電端]              [傳輸端]         [配電端]         [備援端]
SOFC現場發電    →    HVDC高壓直流  →  智慧PDU分配  →  BBU不斷電
（中興電/康舒）       （台達電/光寶）   （研華/健和興）   （台達電/AES）

                    GaN功率元件（嘉晶/漢磊）——貫穿各環節的核心零件
```
