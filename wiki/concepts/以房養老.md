# 以房養老（不動產逆向抵押貸款）

**最後更新：** 2026-06-12
**相關集數：** 2026061202

---

## 概述

以房養老是「不動產逆向抵押貸款」的俗稱。與一般房貸相反：不是「貸款買房、每月還款」，而是「已擁房屋、每月領取生活費」。銀行依照房屋估值核定貸款額度，每月分批撥款給借款人，借款人過世或賣屋時再一次清償貸款。2026年Q1新增申請件數創歷史新高（323件），累積核貸1.6萬件。

---

## 五大迷思澄清（蘇律師，2026-06-12 中集）

| 迷思 | 真相 |
|------|------|
| 房子會被銀行收走 | 否。房屋仍是自己的，銀行只是抵押權人，非所有權人 |
| 以房養老就是把房子賣給銀行 | 否。只是設定抵押權，房子依然屬於遺產 |
| 貸款到期會被趕走 | 若保守試算（3000萬×50%=1500萬，月領4萬×30年），仍有殘值，可申請展延 |
| 兄弟姊妹會反對 | 無法反對（法律上父母有決定權）；解法：若反對，請他們給生活費 |
| 死後房子歸銀行 | 否。房子仍為遺產，先清償債務，剩餘價值可由繼承人繼承或捐公益 |

## 試算範例（65歲，屋齡30年，月領4萬元，利率2.5%，30年期）

| 項目 | 數值 |
|------|------|
| 房屋市值（假設） | 3000萬元 |
| 貸款額度（約50%） | 1440萬元 |
| 月領金額（初期） | 4萬元 |
| 12年後實領（因利息遞增） | 約2.8萬元 |
| 貸款費用 | 約5000元+查詢費300元 |

## 優點（蘇律師分析）

1. **有尊嚴**：不用向子女伸手要錢
2. **降低詐騙風險**：銀行抵押後，詐騙集團難以一次騙走大筆款項
3. **節稅效果**：貸款作為負債可於遺產稅中扣除
4. **安居不必搬家**：留在自住房屋持續生活

---

## 適合族群

- 60歲以上長輩、自住房無房貸者
- 退休準備不足、缺乏現金流者
- 頂客族/單身族（無子女繼承考量）
- 房屋位於都市蛋黃區（鑑價較高）
